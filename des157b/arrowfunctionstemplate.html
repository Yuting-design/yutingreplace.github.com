Arrow Functions

This function accepts a string as parameter. It will return "Hello" + "parameter".

More learning resources: MDN

Named Function

            function(parameter){
                Do something();
            }

Function Expression

            const sayHello = function(parameter){
                return `Hello ${parameter}`;
            }

Arrow Function

            //Basic
            const sayHello = (parameter) => {
                return `Hello ${parameter}`;
            } 

            //if one parameter, you can removing the parentheses.
            const sayHello = parameter => {
                return `Hello ${parameter}`;
            }

            //if one parameter, contains only one statement, and the function returns a value,
              you can also remove the curly braces and the return keyword.
              const sayHello = parameter => `Hello ${parameter}`;
