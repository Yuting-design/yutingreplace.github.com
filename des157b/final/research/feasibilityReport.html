Feasibility Report

Features

My final project, "Today is", shows how each day is special to people by collecting data and converting it into visual information. Users can enter and submit why today is a special day. Their information will appear on the web page and will be colored according to what is typed. Users can view information uploaded by others and see more details by clicking on the title.

"Today is" includes the ability to display dates, collect data, display data, and information and color conversion.

Technologies

Display dates

The Display Date function shows the user the date of the day, including word month and numeric date.

Although I have never tried to capture and display time in other peoject, I do not think it is difficult after investigating. I can get the time using Date() and the ECMAScript Internationalization API. I can then write the data to the document using innerHTML().

Collect data

Collect Data collects the characters or options uploaded by the user.

I'll use the HTML form element to get data and then read it in javascript. I don't expect this to be too difficult, as I did this exercise in an earlier FrendList assignment.

Display data

The Display data function displays the uploaded data in an orderly manner on the web page.

After retrieving data from a table, Display Data writes the data to an HTML file in a specific format to Display on the page. It calls the Get Color function to get a color data.Javascript also needs to work with CSS to arrange the different "days". This may require the application of grid or Flex Box. I might need to use library to arrange the boxes. The optional library is Masonry.

Information and color conversion

In this section, I plan to connect the data uploaded by users with colors.

There are two options. Plan A: I use the emotion recognition API to distinguish the emotional tendencies of the data, and then set different tendencies to different colors. The optional APIs are Sentiment Analysis API and Text to Emotion API. Plan B: I convert user data directly to color and display it in style. This is easier because I've managed to do this with Library Liberty.
