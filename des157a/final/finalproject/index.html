Game Rules

Help Paul or Grace win the game. The piggy whose turn it is rolls the dice. The total of the roll is added to the current piggy's score, unless either die comes up as a "one". If this happens, this piggy's turn is over, and it is the other piggy’s turn. After each roll, the current piggy can either roll again, (assuming a "one" was not rolled) or if the current player feels that luck is running thin, they can pass to the other player. The first piggy to get 30 points or higher wins.

Oh, and if a piggy rolls two "ones" (snake eyes), the piggy current score gets zeroed out. So don’t do that.
